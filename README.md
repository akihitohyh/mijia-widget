# 米家桌面小插件

一个基于 PyQt6 的 Windows 桌面小插件，用于显示米家账号下的所有设备状态和智能插座用电信息。

## 功能特性

- 🔗 自动同步米家账号设备
- 🟢 显示设备在线/离线状态
- 🏠 显示设备所属房间
- 📦 支持所有米家智能设备
- ⏰ 定时自动刷新（默认30秒）
- 🖱️ 窗口可拖拽移动，可调整大小
- 🎨 深色主题UI，支持自定义颜色
- 🔌 智能插座用电信息显示（功率、今日用电、累计用电）
- 👁️ 点击穿透模式（鼠标穿透窗口操作桌面）
- 📌 窗口置顶显示
- 🔔 系统托盘图标支持
- 📱 图形化扫码登录/退出登录

## 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10 / Windows 11 |
| Python | 3.8 或更高版本 |
| 网络 | 需要联网访问米家服务器 |
| 账号 | 小米/米家账号 |

## 依赖要求

### 运行依赖

| 依赖包 | 版本 | 说明 |
|--------|------|------|
| mijiaAPI | >=3.0.0 | 米家API接口 |
| PyQt6 | >=6.4.0 | GUI界面框架 |
| qasync | >=0.23.0 | 异步支持 |
| qrcode | >=7.0 | 二维码生成（扫码登录） |
| pillow | >=9.0 | 图像处理 |

### 打包依赖（可选）

| 依赖包 | 版本 | 说明 |
|--------|------|------|
| pyinstaller | >=6.0 | 打包为exe |

## 安装步骤

### 方式一：Python源码运行（推荐）

1. **克隆或下载项目**
   ```bash
   cd c:\pythonp\mijia
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   # 或手动安装
   pip install mijiaAPI>=3.0.0 PyQt6>=6.4.0 qasync>=0.23.0 qrcode>=7.0 pillow>=9.0
   ```

3. **首次登录米家账号**
   ```bash
   mijiaAPI --login
   ```
   按提示扫描二维码，用米家APP扫描完成登录。

4. **启动插件**
   ```bash
   python main_widget.py
   # 或双击运行
   start.bat
   ```

### 方式二：EXE试用版（实验性）

> ⚠️ **注意**：EXE格式目前为试用版，可能存在功能限制或不稳定。建议优先使用Python源码运行。

1. **下载或生成EXE**
   - 从Release页面下载预编译的EXE文件，或
   - 自行打包（见下方打包说明）

2. **运行EXE**
   ```bash
   # 直接双击运行
   dist\米家桌面插件.exe
   ```

3. **登录账号**
   - 点击设置 → 扫码登录
   - 或在命令行运行 `mijiaAPI --login`

## 打包为EXE（可选）

如需自行打包为EXE文件：

```bash
# 安装打包工具
pip install pyinstaller

# 执行打包命令
python -m PyInstaller --name "米家桌面插件" --onefile --windowed --clean --runtime-hook hook_stdio.py main_widget.py

# 或使用打包脚本
double-click build.bat
```

打包完成后，可执行文件位于 `dist\米家桌面插件.exe`。

## 使用说明

### 界面操作

| 按钮 | 功能 |
|------|------|
| `—` | 最小化到系统托盘 |
| `📌`/`📍` | 切换窗口置顶 |
| `👁` | 开启/关闭点击穿透 |
| `⚙` | 打开设置面板 |
| `×` | 关闭（实际最小化到托盘） |

### 系统托盘菜单

右键点击托盘图标可打开菜单：
- **显示窗口** - 恢复窗口显示
- **置顶显示** - 切换置顶状态
- **刷新设备** - 手动刷新设备列表
- **设置** - 打开设置面板
- **退出** - 完全退出程序

### 智能插座选项

点击插座卡片的"选项"按钮，可设置显示哪些信息：
- 当前功率 (W)
- 今日用电 (度)
- 累计用电 (度)
- 开关状态

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| ESC | 最小化到托盘 |
| 鼠标拖动 | 移动窗口位置 |
| 鼠标拖拽边缘 | 调整窗口大小 |

### 配置文件

配置文件位置：`~/.config/mijia-api/widget_config.json`

编辑 `config.py` 可修改默认配置：

```python
REFRESH_INTERVAL = 30   # 设备刷新间隔（秒）
WINDOW_WIDTH = 300      # 窗口宽度
WINDOW_HEIGHT = 400     # 窗口高度
```

## 常见问题

### 1. 首次使用无法显示设备

**解决**：需要先登录米家账号
```bash
mijiaAPI --login
# 或使用 登录.bat
```

### 2. 二维码登录一直显示"正在获取二维码"

**解决**：使用命令行登录
```bash
# 运行登录辅助脚本
登录.bat
# 或在命令行运行
mijiaAPI --login
```

### 3. 设备列表为空

**解决**：
1. 检查网络连接
2. 重新登录
```bash
# 删除认证文件后重新登录
del %USERPROFILE%\.config\mijia-api\auth.json
mijiaAPI --login
```

### 4. EXE版本启动报错

**解决**：
1. 确保已使用Python版成功登录过一次
2. 检查杀毒软件是否拦截
3. 尝试使用管理员权限运行
4. 建议改用Python源码运行

### 5. 用电数据显示异常

**注意**：
- 用电量数据需要设备支持MIOT协议
- 累计用电统计的是最近30天的数据
- 云端统计数据可能有延迟

## 注意事项

1. **首次使用必须先登录** - 运行 `mijiaAPI --login` 或点击设置中的扫码登录
2. **登录信息保存位置** - `~/.config/mijia-api/auth.json`
3. **EXE版本限制** - 试用版可能存在功能限制，建议优先使用Python源码
4. **用电数据** - 部分旧型号设备可能不支持获取详细用电数据
5. **网络要求** - 需要稳定的网络连接才能正常获取设备状态

## 技术栈
