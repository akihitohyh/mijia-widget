# 自制米家桌面小插件，实时显示设备状态｜开源分享

> 这是一个基于 PyQt6 开发的 Windows 桌面小工具，可以实时显示米家账号下的所有设备状态，还支持智能插座用电信息监控。

*(建议在这里放一张软件实际运行的截图)*

---

## 为什么要做这个？

作为一个智能家居爱好者，家里的米家设备越来越多，但每次想看设备状态都要打开手机App，感觉很麻烦。于是就想做一个**桌面小插件**，可以**实时显示**所有设备的状态，还能**监控智能插座的用电情况**。

**核心需求：**
- 一眼看到所有设备在线/离线状态
- 实时显示智能插座功率和用电量
- 窗口置顶，随时可见
- 体积小，不占用太多资源

---

## 功能亮点

### 1. 设备状态实时监控
- 自动同步米家账号下的所有设备
- 显示在线/离线状态（绿色/红色指示灯）
- 显示设备所属房间
- 每30秒自动刷新

### 2. 智能插座用电信息
- **当前功率**：实时显示瓦数
- **今日用电**：用了多少度电
- **累计用电**：近30天总用电统计
- **开关状态**：显示插座开启/关闭

### 3. 贴心小功能
- **点击穿透模式**：开启后鼠标可以穿透窗口操作桌面，再也不怕窗口挡事
- **系统托盘图标**：最小化到托盘，不占用任务栏
- **窗口置顶**：始终显示在最上层
- **可调整大小**：拖拽边缘改变窗口大小

## 安装使用

### 准备工作
- Windows 10/11 系统
- Python 3.8+（如果没装过Python可以参考网上教程）

### 三步搞定

**第一步：下载代码**
```bash
git clone https://github.com/akihitohyh/mijia-widget.git
cd mijia-widget
```

**第二步：安装依赖**
```bash
pip install -r requirements.txt
```

**第三步：双击启动**
直接双击 `启动插件.bat`，首次使用会引导登录，之后一键启动！

### 登录方式
程序提供两种登录方式：
1. **扫码登录（推荐）**：弹出二维码，用米家App扫描
2. **命令行登录**：在终端显示二维码链接

## 常见问题

**Q1：为什么显示"共0个设备"？**
A：需要先登录米家账号，首次运行 `启动插件.bat` 会引导登录。

**Q2：用电数据不准确？**
A：用电量统计的是云端数据，可能有延迟。部分旧型号设备可能不支持详细用电数据。

**Q3：可以打包成EXE吗？**
A：可以，运行 `build.bat` 即可自动生成EXE文件。但EXE是实验性功能，建议优先使用Python源码运行。
